Style Guide

Navigation Button

Font for navigation button in smartwatch is 16px. Font for navigation button in smartphone and game console is 22px.

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Default Button</a>

Colors:

  • Border:

    #DF9C58

  • Background:

    #D18334

  • Font:

    #FFFFFF

  • Hover:

    #CC6600

  • Active:

    #DF9C58

Rendered element:

Default Button

In-text link

In-text link is in Avenir Book font and 16px for smartwatch and smartphone. Avenir Medium and 20px for game console.

CSS selector:

.link

Sample code:

<a class="link" href="">Sample Link</a>

Rendered element:

Sample Link

Text Input Field and Label

CSS selectors:

.label

.inputfield

Sample Code:

<label class="label">Default Label</label>

<input type="text" class="inputfield" name="lastname">

Labels are in Avenir Book font. 16px for smartwatch and smartphone, and 20px for game console.

Rendered element:

18px

Avenir Book

Headings

The font for the headings is Avenir. The thickness of the font represents the hierarchy of the overall structure of the website.

CSS selectors:

.pageheading

.subpageheading

.sectionheading

Colors:

  • Background:

    #FFC287

  • Font:

    #FFFFFF

Sample code for Page Heading:

<h1>Page Heading</h1>

Rendered Element:

Page Heading

32px

Avenir Heavy

Sample code for Subpage Heading:

<h2>Subpage Heading</h2>

Rendered Element:

Subpage Heading

26px

Avenir Medium

Sample code for Section Heading:

<h3>Section Heading</h3>

Rendered Element:

Section Heading

22px

Avenir Medium

Paragraphs

Font is Avenir Light. Default size for body text is 18px.

CSS selector:

.paragraph

Sample code for Paragraphs:

<p class="paragraph">Lorem ipsum dolor sit amet.</p>

Sample Paragraph:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean euismod bibendum laoreet. Proin gravida dolor sit amet lacus accumsan et viverra justo commodo.

Bulleted Lists

Items in bulleted lists for smartwatch and smartphone are 14px. Items in bulleted lists for game console are 18px.

CSS selector:

.bulleted-list

Sample code for Bulleted List:

<ul class="bulleted-list">
<li>Rule 1</li>
<li>Rule 2</li>
<li>Rule 3</li>
</ul>

Sample Bulleted List:

  • Rule 1
  • Rule 2
  • Rule 3
